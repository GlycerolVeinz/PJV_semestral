Goblin OverFlow

When turned on in console question will appear: "Launch Goblin OverFlow? [y/n]"

If launched user is greated by window where there are these elements:

- Dungeon map
- Buttons
    - add entity
    - save / load
    - start / pause

save / load buttons make pop-ups with possibility to change saving / loading file (directory)

Goal of the game is to get rid of all the Goblins in the Dungeon, by adding entities.
(number of entities is limited)
(entities have hard time living with each other)
(goblins multiply faast)
